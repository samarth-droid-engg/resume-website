# Resume Website (HTML Only)

This is a simple single-page resume website built using only HTML, without any CSS or external libraries.  
The goal of this project is to practice clean structure, semantic HTML tags, and proper layout formatting using headings, paragraphs, lists, tables, and sections.

---

# Features

- Header with name and title  
- About section  
- Skills list  
- Experience table  
- Projects section  
- Education table  
- Contact details  
- Clean and readable structure with proper HTML semantics  
- No CSS used

---

# How to Use

1. Download or clone this repository.
2. Open the `index.html` file in any modern browser.
3. That’s it — the page will load instantly without any setup.
